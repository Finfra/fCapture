# TOC Generator (목차 생성)

마크다운 파일에 대한 목차(Table of Contents)를 자동으로 생성합니다.

## 사용법

대상 마크다운 파일을 읽어서 헤더를 수집하고 목차를 생성합니다.

## 로직
1. `## 📋 목차` 헤더를 찾습니다.
2. 문서의 모든 `H1` (`# 제목`) 헤더를 수집합니다.
3. `1. [제목](#앵커)` 형식의 목록을 생성합니다.
4. 기존 목차를 새 목차로 교체합니다.
