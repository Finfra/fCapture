# 용어
```
window_active   : 현재 최상단 윈도우 캡처 (피드백 없음)
window_flash    : 현재 최상단 윈도우 캡처 + 0.5초 파란 테두리 flash (어느 윈도우 찍혔는지 확인용)
window_pointer  : 마우스 커서 위치의 윈도우 캡처
region_user     : 마우스 드래그 영역 선택 캡처
region_static   : JSON staticRegion 좌표 고정 캡처
```
